# BitSizeI function

Namespace: Microsoft.Quantum.Math

```qsharp
function BitSizeI(a : Int) : Int
```

## Summary
For a non-negative integer `a`, returns the number of bits required to represent `a`.
NOTE: This function returns the smallest n such that a < 2^n.
