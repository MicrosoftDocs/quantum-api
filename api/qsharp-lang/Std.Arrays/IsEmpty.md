# IsEmpty function

Fully qualified name: Std.Arrays.IsEmpty

```qsharp
function IsEmpty<'T>(array : 'T[]) : Bool
```

## Summary
Returns true if and only if an array is empty.

## Input
### array
The array to be checked.

## Output
`true` if and only if the array is empty (has length 0).
