# X operation

Fully qualified name: Std.Intrinsic.X

```qsharp
operation X(qubit : Qubit) : Unit is Adj + Ctl
```

## Summary
Applies the Pauli _X_ gate.

## Input
### qubit
Qubit to which the gate should be applied.
