# Z operation

Fully qualified name: Std.Intrinsic.Z

```qsharp
operation Z(qubit : Qubit) : Unit is Adj + Ctl
```

## Summary
Applies the Pauli _Z_ gate.

## Input
### qubit
Qubit to which the gate should be applied.

## Remarks
$$
\begin{align}
    \sigma_z \mathrel{:=}
    \begin{bmatrix}
        1 & 0 \\\\
        0 & -1
    \end{bmatrix}.
\end{align}
$$
