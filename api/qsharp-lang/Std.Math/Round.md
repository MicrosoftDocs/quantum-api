# Round function

Fully qualified name: Std.Math.Round

```qsharp
function Round(value : Double) : Int
```

## Summary
Returns the nearest integer to the specified number. Half is rounded towards zero.
For example: Round(3.7) = 4; Round(-3.7) = -4; Round(3.5) = 3;
